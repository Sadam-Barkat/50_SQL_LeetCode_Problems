# LeetCode 1667: Fix Names in a Table

## Problem Statement
Given a table `Users` with a column `name`, write an SQL query to format each name such that:  
- The first letter is **uppercase**.
- The remaining letters are **lowercase**.

### **Table: Users**
| user_id | name  |
|---------|-------|
| 1       | aLiCe |
| 2       | boB   |
| 3       | JoHn  |

### **Expected Output**
| name  |
|-------|
| Alice |
| Bob   |
| John  |
